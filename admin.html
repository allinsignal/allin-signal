Admin Tools

Maintenance tools for the Allin Signal database.

SMA Backfill — Single Ticker
Fetches 10 years of price history from Yahoo Finance and recomputes the 200-week SMA for one ticker. Use this when a stock is missing SMA history. Takes 2–5 seconds per ticker.

Run Daily Scan
Fetches today's market data from Polygon, runs the signal model on all tickers, applies the confidence gate, updates rankings, and syncs ETF flags. Normally runs automatically after market close — use this to trigger it manually.

Price History Backfill — Polygon
Fetches 5 years of daily bars from Polygon for any ticker with insufficient price history. Run this after adding new tickers (like ETFs) so sparklines and YTD returns show correctly. Takes 15–30 seconds.
